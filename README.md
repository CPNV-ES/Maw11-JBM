# MAW-1-JBM

## Description

App for creating exercises for students

## Getting Started

### Prerequisites

* PHP 8.4
* Composer 2.8
* Xdebug 3.4
* IDE tested : PhpStorm and Cursor
* OS supported : Windows 11 and Fedora 42

### Configuration

```shell
    composer install
```
